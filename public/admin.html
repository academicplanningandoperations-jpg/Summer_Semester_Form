📤 Upload Student Eligibility Data (CSV)

Upload a flat CSV file containing one row per eligible course per student.
Required columns: Student Global ID, Student Name, Student Email, School Name, Program Name, Program Code, Semester, Course Code, Module Name, Credit Point, Final Grade
Category is auto-derived from Final Grade: F = Failed, AB = Debarred, all others = Improvement.
Column headers are flexible — variants like SAP_ID, Credits, Course_Name etc. are also matched.
